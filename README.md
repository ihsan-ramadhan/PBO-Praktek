# Pemrograman Berorientasi Objek - Praktikum

Repositori ini berisi kumpulan tugas praktikum dan proyek untuk mata kuliah Pemrograman Berorientasi Objek (PBO). Fokus utama dari praktikum ini adalah implementasi konsep OOP menggunakan bahasa pemrograman Java.

## Struktur Direktori
Repositori ini disusun berdasarkan minggu praktikum:
* **W1**: Dasar-dasar Java (Data Types, Strings, Constants)
* **W2 - W4**: Pemodelan Objek, Enkapsulasi, dan Pewarisan (Inheritance)
* **W5 - W6**: Polimorfisme, Interface, dan Abstraksi
* **W9**: Penanganan Eksepsi (Exception Handling)
* **W10 - W11**: Generic Programming, Collection, dan GUI dasar (Point of Sales)

## Prasyarat
* Java Development Kit (JDK) 8 atau versi terbaru.
* Editor/IDE (VS Code, IntelliJ IDEA, atau Eclipse).

## Cara Menjalankan
1. Clone repositori ini.
   
2. Navigasi ke folder minggu tertentu, misalnya W1.

3. Kompilasi file: javac NamaFile.java

4. Jalankan program: java NamaFile
